## Hackathon Tracks

This project is submitted to the **0G / Open Agents Hackathon** for:

**Track 1 — Best Agent Framework, Tooling & Core Extensions**
MemoryMerge is a reusable SDK and OpenClaw skill that adds 
persistent decentralized memory to any agent framework.
Drop-in installation. Three lines of code.

**Track 2 — Best Autonomous Agents, Swarms & iNFT Innovations**
A live multi-agent swarm (Planner + Researcher + Critic) 
that coordinates exclusively through shared 0G Storage state,
with autonomous reflection loops powered by 0G Compute.
